About This Site

Web atau blog ini terdiri dari 6 bagian

Bagian Pertama

FutScrimmage, web atau blog ini dibentuk pada tanggal 8 Agustus 2020, dikamar kos saya sendiri tepatnya ketika tim idola saya, Real Madrid dikalahkan oleh salah satu klub Liga Inggris, Manchester City.

Bagian Kedua

Setelah laga tersebut, saya mencari artikel-artikel atau tulisan pribadi para fans yang dituangkan melalui blog ataupun web pribadi. namun yang saya temukan sangat minim, kebanyakan dari web jurnalisme sepakbola bonafite seperti panditfootball.com

Bagian Ketiga

Hadirnya web ini tentu tidak ada tujuan sama sekali untuk menyaingi web jurnalisme sepakbola bonafide seperti panditfootball.com. justru tujuan utama saya membuat web ini adalah untuk menambah "ikan" dalam ekosistem "kolam" jurnalistik sepakbola lokal.

Bagian Keempat

Inspirasi saya mendirikan web ini berasal dari web-web jurnalistik sepakbola besar baik lokal maupun luar negeri, pendapat para pandit sepakbola, atau komentator sepakbola. Terkhusus saya sangat terinspirasi dengan figur seperti Pangeran Siahaan, Tyo Prasetyo Utomo, Dex Glenniza dan Coach Justin. Baik cuitan ataupun tulisan yang mereka tuangkan, merupakan hal yang asik untuk dibaca.

Bagian Kelima

Garis besar Web atau Blog ini sendiri tepatnya berisikan opini pribadi penulis, dalam hal ini saya sendiri. Dibagi menjadi dua artikel, artikel pertama berisi opini-opini mengenai sepakbola lokal artikel kedua berisi opini-opini mengenai sepakbola luar negeri.

Bagian Keeenam

Tentu masih ada ketidaksempurnaan dalam pembuatan web ini, saya sendiri akan berusaha semaksimal mungkin untuk memperbaiki, melengkapi, dan membuat web ini semakin menarik kedepannya. Tak lupa saya haturkan ucapan terimakasih kepada Dicoding yang mau membagi ilmunya
